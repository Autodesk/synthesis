Steps to install Synthesis: an Autodesk Technology for OSX ,

1. Unzip the macos installer that was retrieved from the website
2. Double click the Synthesis.dmg file or appropriately named equivalent
3. Open a command terminal and browse to the unzipped directory where synthesis was downloaded
4. run ./post-install.sh -install

Steps to uninstall Synthesis: an Autodesk Technology for OSX ,

1. Unzip the macos installer that was retrieved from the website
2. Open a command terminal and browse to the unzipped directory where synthesis was downloaded
3. run ./post-install.sh -uninstall
4. Open your Dashboard and click and drag Synthesis into the trash bin
